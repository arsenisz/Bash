## Розділ 2: Написання скриптів мовою Bash

### Завдання
Скрипт читає параметр (формат файлів: txt / dev / pdf) і копіює всі файли вказаного формату з поточної директорії в `/tmp/newdir`

```bash

```

## Контрольні питання

### 1. Призначення Bash, структура скрипта
Bash — командний інтерпретатор і мова сценаріїв в Linux. Використовується для автоматизації задач, керування файлами та адміністрування системи.
```bash
#!/bin/bash      # shebang
# коментар
VARIABLE="value" # змінна
command          # команда
```

### 2. Змінні в Bash
Присвоєння без пробілів навколо `=`:
```bash
NAME="Arsen"   # рядкова змінна
NUMBER=42      # числова змінна
echo "$NAME"   # виведення
```
Спеціальні: `$1, $2` — аргументи, `$0` — ім'я скрипта, `$#` — кількість аргументів, `$?` — код повернення.

### 3. Введення з клавіатури і виведення в консоль
```bash
echo "Привіт!"                 # виведення
read -p "Введіть ім'я: " NAME  # введення з підказкою
```

### 4. Умови в Bash
```bash
if [ умова ]; then
    # якщо істина
elif [ умова ]; then
    # інакше якщо
else
    # інакше
fi
```
Оператори: `-z` (порожній рядок), `-f` (файл існує), `-d` (директорія), `-eq`, `-ne`, `-lt`, `-gt`.

### 5. Цикли в Bash
```bash
for i in 1 2 3; do echo "$i"; done          # for по списку
for ((i=0; i<5; i++)); do echo "$i"; done   # C-style
while [ умова ]; do команда; done           # while
until [ умова ]; do команда; done           # until
```

### 6. Написання скриптів на Bash
1. Створити файл `.sh`
2. Перший рядок — `#!/bin/bash`
3. Написати команди
4. Надати права: `chmod +x script.sh`
5. Запустити: `./script.sh`
